## 下载

### Docker

```markup
docker pull apachecn0/ds-ipynb-zh
docker run -tid -p <port>:80 apachecn0/ds-ipynb-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install ds-ipynb-zh
ds-ipynb-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g ds-ipynb-zh
ds-ipynb-zh <port>
# 访问 http://localhost:{port} 查看文档
```
